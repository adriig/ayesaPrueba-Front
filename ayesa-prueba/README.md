## Installation

Clone the repository
```git
    git clone https://github.com/adriig/ayesaPrueba-Front.git
```

Install the repository

```bash
  npm install
  cd my-project
```
## Deployment

#### Front Deployment

To deploy this project run
```bash
  ng serve --o
```
## Tech Stack

**Client:** Angular

**Server:** Node, Express, Docker
